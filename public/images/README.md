# Logo Instructions

Please place your company logo file here with the name `logo.png`

The logo should be:
- PNG format (or JPG/JPEG - update the file extension in the code if using JPG)
- Recommended size: 200x200 pixels or larger
- Transparent background preferred
- High resolution for best quality

The logo will be displayed in:
- Website navigation header
- Invoice headers
- Print views

If the logo file is not found, the website will fall back to text-only display.
